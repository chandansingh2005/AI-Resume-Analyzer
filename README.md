# AI Resume Analyzer

AI Resume Analyzer is a full-stack web application that helps users analyze their resumes using Artificial Intelligence. Users can upload PDF resumes, extract resume content, and receive detailed ATS-based feedback powered by Google Gemini AI.

## Features

* User Authentication (Register, Login, Logout)
* JWT-based Protected Routes
* User Profile Management
* Resume Upload (PDF Only)
* PDF Text Extraction
* AI-Powered Resume Analysis using Gemini AI
* ATS Score Generation
* Resume Summary
* Strengths & Weaknesses Analysis
* Missing Skills Detection
* Recommended Job Roles
* Keyword Analysis
* Personalized Improvement Suggestions
* Resume Analysis Stored in MongoDB
